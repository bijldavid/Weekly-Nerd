Weekly Nerds Goals My work

Doelen

Doel: 1

Beter worden in javascript schrijven

Toelichting

Ik wil graag beter worden in JavaScript. Ik denk dat ik de basis redelijk goed begrijp, maar ik vind het nog lastig om wat complexere dingen zelf te bouwen. Het lijkt me een handige skill om onder de knie te krijgen.

Ik ben tevreden wanneer ik

Zelfstandig een stuk JavaScript-code heb geschreven, goed begrijp wat ik heb gemaakt, en het zonder hulp van AI of anderen kan toelichten en verbeteren.

Doel: 2

Minder snel AI code te laten generenen

Toelichting

Dit doel hangt best goed samen met mijn eerste doel. Ik ben vaak te veel gefocust op het eindresultaat, en dan gebruik ik AI om dingen op te lossen die ik eigenlijk zelf zou moeten leren. Daardoor leer ik het uiteindelijk niet echt goed.

Ik ben tevreden wanneer ik

GEEN stukken gegenereerde code tussen mijn code heb staan. Ik ga proberen zelfstandig problemen op te lossen voordat ik AI raadpleeg. Als ik AI gebruik, is dat alleen om uitleg te vragen, niet om complete code over te nemen.

Doel: 3

Niet automatisch akkoord gaan met ideeën van andere

Toelichting

Ik vind mezelf niet super sterk in conceptueel denken. In een groep laat ik het daarom vaak aan anderen over. Ik ga soms te snel akkoord met ideeën, ook als ik eigenlijk denk dat ze niet zo goed zijn. Ik wil leren om vaker mijn mening te geven.

Ik ben tevreden wanneer ik

Vaker initiatief neem in gesprekken, mijn eigen ideeën durf te delen, en actief heb meegedacht bij groepsbesluiten.
